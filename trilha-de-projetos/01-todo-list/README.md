- ➕ **Adicionar, editar e excluir tarefas**  
- 📌 Campo de **observações** para detalhar cada atividade  
- ⏰ Definir **data/hora limite** para execução  
- 🔔 **Alarmes sonoros e visuais (Zoom, Piscante, Pulse)**  
- 🎵 **Upload de áudio personalizado** para alarmes  
- 🔕 Botão para parar testes de som ou alarmes ativos  
- 📊 **Barra de progresso dinâmica**  
- 💾 Salvamento automático no **LocalStorage**  
- 🌙 Alternar entre **modo Dark/Light** com persistência
